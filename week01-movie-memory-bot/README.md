# Movie Recommendation Bot with Memory

AI Agents 마스터클래스 #2.3 — 사용자 **취향(좋아하는 장르)** 과 **이미 본 영화**를 기억하는 영화 추천 챗봇 (프레임워크 없음).

## 무엇

기본 호출은 stateless. 해결책은 단순 — 대화를 전부 `messages` 리스트에 누적하고 매번 통째로 보낸다. 메모리 덕에 봇이:

- 좋아하는 장르(SF)를 기억해 그에 맞춰 추천
- 이미 본 영화(인셉션·인터스텔라)는 추천에서 제외
- 나중에 "내 취향/본 영화 뭐랬지?" 회상

모델: `gpt-4o-mini`

## 4턴 시나리오

1. `"나는 SF 영화를 좋아해"`
2. `"인셉션이랑 인터스텔라는 이미 봤어"`
3. `"오늘 밤에 뭐 볼지 추천해 줄래?"` → SF + 안 본 영화로 추천
4. `"내가 좋아하는 장르랑 이미 본 영화가 뭐라고 했지?"` → 메모리로 회상

## 실행

```bash
uv sync          # .venv 생성 + 패키지 설치
# .env 는 movie-agent 키 재사용해 이미 들어있음
```

`main.ipynb` 열고 위에서부터 TODO 채우며 진행 → **Run All** 후 출력 찍힌 상태로 커밋 제출.
